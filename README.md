# Atlas DePIN: Scaling AI Compute from Algeria 🇩🇿

Atlas DePIN is a decentralized computing project based in Batna, Algeria. Our goal is to democratize access to AI compute power by deploying high-performance nodes on decentralized protocols.

## 🚀 Mission
Breaking the centralization bottleneck of AI infrastructure. We leverage competitive energy costs and next-gen fiber connectivity in Algeria to provide low-latency, high-availability compute power to the global DePIN ecosystem.

## 🛠 Tech Stack
- **Orchestration:** Kubernetes (K8s) for high-availability.
- **Environment:** Docker containers for scalable AI workloads.
- **Monitoring:** Prometheus & Grafana for real-time transparency.
- **Hardware:** Optimized for performance-per-watt.
